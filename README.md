# BPCL Electronic Totalizer Receipt OCR

## Overview

BPCL Electronic Totalizer Receipt OCR is a professional Java desktop application for scanning, extracting, validating and comparing Electronic Totalizer (ET) receipts generated by Bharat Petroleum fuel dispensing units.

The application uses OpenCV for image preprocessing and Tesseract OCR for text recognition, followed by a custom BPCL receipt parser to extract nozzle-wise readings accurately.

The project is being developed with a clean layered architecture and enterprise coding standards.

---

## Objectives

- Read BPCL Electronic Totalizer receipt images
- Improve image quality using OpenCV
- Extract text using Tesseract OCR
- Parse ET receipt values
- Compare two receipts
- Generate Difference Reports
- Export Excel and PDF reports
- Provide an easy-to-use Desktop UI
